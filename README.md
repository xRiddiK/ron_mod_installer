# Ready or Not Mod Loader
## Tech-Stack
- Framework: [Electron](https://www.electronjs.org/)
- UI: [TailwindCSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/)
- Build & Packaging: [Electron Builder](https://www.electron.build/)
- File Handling: [FS-Extra](https://github.com/jprichardson/node-fs-extra)
- Process Control: Node.js child_process

---

## Features

- Mod Browser (Mods werden von [jxcs.space](https://jxcs.space) gefetched)
- Installierte Mods verwalten
- Game Detection (Erkennt automatisch, ob Ready or Not läuft)
- Launch Integration (Starte das Spiel direkt im Launcher)
- "Radio" (mir war langweilig)
- Einfacher Folder-Selector für den Ready or Not Pfad (automatische Validierung auf ReadyOrNot/Content/Paks)
